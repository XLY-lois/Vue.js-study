### v-model 双向数据绑定
- v-bind只能实现数据从m到v的单向绑定 无法实现双向绑定
- v-model v的数据改变，就能同步到m里
```
<body>
    <div id="app">
    <h4>{{ msg }}</h4>
    <input type="text" v-model="msg">
    </div>
</body>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            msg:'大家都是好孩子'
        },
    });
</script>
```
